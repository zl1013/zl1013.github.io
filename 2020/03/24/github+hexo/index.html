Github+HEXO (win)

Github+HEXO (win)

  • 安装git

  • 安装nodejs

  • 本地创建Hexo存储文件夹

  • 文件夹内git bash here

  • 可通过cmd命令移动到文件夹，也可以在文件夹地址栏直接输入cmd

  • 安装Hexo

    npm install -g hexo-cli3

安装完成，可能会有WARN，但不会影响正常使用

npm install hexo --save

Hexo安装完成，检测是否正确安装

hexo -v

  • Hexo的配置

  • 在当前目录下新建一个名为blog的文件夹，用于存放博客网站的信息
    命令行进入blog，初始化该文件夹，并安装所需组件

    hexo init 
npm install

  • 安装完成后，检测是否安装成功

    hexo g
hexo s

  • 根据提示访问http://localhost:4000/(注意http不是https)

  • 将GitHub Page与Hexo关联

  • 配置Git个人信息
    这一步骤是为了以后我们用git命令提交不再需要繁琐的输入我们的Github帐号和密码。

  • 在blog 目录下，执行下列命令配置你的用户名

    git config --global user.name "username"
git config --global user.email "emailm"
git config --list  //查看用户信息

  • 生成秘钥

    ssh-keygen -t rsa -C "email"

  • 生成文件默认路径

    C:\Users\zzl\.ssh

  • 输入eval “$(ssh-agent -s)”，添加密钥到ssh-agent

    eval "$(ssh-agent -s)"

  • 再次输入命令

    ssh-add ~/.ssh/id_rsa

  • 添加帐号SSH Keys
    登录Github，点击头像下的settings，添加ssh

  • 输入下列命令进行验证是否成功

    ssh -T git@github.com

接下来Hexo踩坑开始！！！
